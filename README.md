# Rent CMS - Property Management System

A comprehensive Content Management System for rental properties with three user roles: Admin, Tenant, and Guest.

## Features

- **Guest Role**: Browse and search rental properties
- **Tenant Role**: Create and manage property listings
- **Admin Role**: Manage users, properties, and system settings

## Technology Stack

- **Backend**: PHP
- **Database**: MySQL
- **Frontend**: HTML5, CSS3, JavaScript
